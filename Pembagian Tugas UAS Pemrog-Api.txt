## Anggota 2 – Authentication (Register + Login + JWT)
Tugas:
- Membuat endpoint:
	POST /api/auth/register
	POST /api/auth/login
- Implementasi:
	bcrypt hashing
	bcrypt compare
	generate JWT (payload: id, email, role)
- Mengembalikan response format sesuai UAS: success, message, data
- Sertakan validasi input (Zod).

Output:
- File app/api/auth/register/route.js
- File app/api/auth/login/route.js
- Validasi auth
- Dokumentasi alur login/register

## Anggota 3 – Middleware + Authorization (Role-based Security Layer)
Tugas:
- Membuat middleware.js
- Memastikan:
	Route /api/auth/* → Public
	Route /api/users/* → Admin only
	Route /api/students/* → User login wajib (Admin juga boleh)
- Meng-inject header:
	x-user-id
	x-user-email
	x-user-role
- Menangani error:
	401 Unauthorized
	403 Forbidden
	
Output:
- File lengkap middleware.js
- Dokumentasi alur keamanan
